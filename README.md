# Lazyman

Um simples e prático To-do list para organização das tarefas diárias!

## 🚀 Começando

### 🔧 Instalação

No terminal NodeJS (utilizado o Visual Studio Code), rode:

```
npm install
```

E:

```
npm run dev
```

## 🛠️ Ferramentas

- [Vite](https://vitejs.dev/);
- [React](https://pt-br.legacy.reactjs.org/);
